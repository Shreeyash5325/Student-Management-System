# Student Management System

## 📌 Description
Java-based Student Management System using File Handling and OOP concepts.

## 🚀 Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student

## 🛠️ Technologies Used
- Java
- File Handling
- Serialization
- ArrayList

## ▶️ How to Run

1. Compile:
   javac com/sms/main/MainApp.java

2. Run:
   java com.sms.main.MainApp
